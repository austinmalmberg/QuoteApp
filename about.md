# Lawn Estimator

Lawn Estimator provides an easy way to estimate the price of mowing properties.

## Features

Core Features

    - Authentication
    - Cost Estimator
    - User configuration
    - Generate ready to print quote 

### User Configation

    1. Gear - for calculating wear and tear
    2. Employees - for calulating wages

### Cost Estimator

    - Wages
    - Travel
        - Gas
        - Mileage
        - Time (man hours)
    - Equipment Costs
        - Wear and Tear
        - Gas
    - Minimum cost (optional)
    - Profit Markup
        - Percentage, or
        - Flat fee
